Telegram-бот для кондитерской (Bakery Bot)

Красивый и функциональный бот на aiogram для заказа тортиков и просмотра меню.

 Что реализовано:
- Главное меню с Reply-кнопками.
- Модульная структура (отдельные хэндлеры для меню и доставки).
- Безопасное хранение токенов через `.env`.
- База данных SQLite3 (`bakery.db`) для хранения информации.

 Как запустить локально:
1. Клонировать репозиторий.
2. Создать файл `.env` и вставить свой токен.
3. Запустить `python main.py`.
